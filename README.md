## 🚀 Local Development

To run this project locally:

git clone [https://github.com/MEGAVARSHINI2004/Megavarshini_portfolio.git]
cd Megavarshini_portfolio
Open index.html in your browser or use a live server
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000
